# UI/UX Keywords Reference

Progressive disclosure reference for mobile UI terminology and adjective palettes, organized for Sleepify.

## Component Keywords

### Navigation (Mobile)
- bottom navigation bar, tab bar
- back button, close button (X)
- header bar, app bar
- drawer menu, hamburger menu
- breadcrumbs, step indicator

### Content Containers
- hero section, featured card
- glassmorphism card, frosted card
- bottom sheet, modal sheet
- accordion, collapsible section
- carousel, horizontal scroll list
- tile grid, masonry grid

### Audio & Player (Sleepify-Specific)
- mini player bar, now playing bar
- play/pause button, stop button
- volume slider, volume ring
- progress bar, seek bar
- waveform visualizer
- equalizer bars, audio spectrum
- mix controls, sound mixer

### Sleep & Wellness (Sleepify-Specific)
- countdown timer, sleep timer
- breathing circle, breath visualizer
- inhale/hold/exhale indicator
- mood chips, mood selector
- bedtime picker, time wheel

### Forms & Inputs
- text input field, search bar
- dropdown, select menu
- toggle switch, checkbox
- slider, range input
- date picker, time picker

### Calls to Action
- primary button (pill-shaped, gradient)
- secondary button (outlined, ghost)
- floating action button (FAB)
- icon button, text link

### Feedback
- toast notification, snackbar
- loading shimmer, skeleton loader
- progress ring, step indicator
- success/error state

## Adjective Palettes

### Calming / Sleep (Default for Sleepify)
- serene, tranquil, ambient
- nocturnal, moonlit, starlit
- soft-glow, gentle, soothing
- muted, subdued, whisper-quiet

### Premium / Luxury
- elegant, refined, exclusive
- glassmorphic, frosted, translucent
- spacious, high-end, curated
- selective contrast, fine lines

### Dark Mode (Always for Sleepify)
- deep backgrounds, near-black canvas
- high-contrast accents, neon highlights
- soft glows, ambient luminescence
- layered surfaces, subtle elevation

### Organic / Nature
- earthy, natural, flowing
- ocean waves, rain drops, forest
- desaturated, warm undertones
- textured, handcrafted feel

## Color Role Terminology

### Backgrounds
- page background, canvas (#0A0E21)
- surface color, card background (rgba white 5-10%)
- overlay, scrim (semi-transparent black)

### Text
- primary text, heading (white #FFFFFF)
- secondary text, body (muted #B0BEC5)
- hint text, placeholder (#546E7A)
- accent text (cyan #00BCD4)

### Accents
- primary accent, brand color (Cyan #00BCD4)
- secondary accent, gradient end (Purple #7C4DFF)
- success (#4CAF50), error (#FF5252), warning (#FFB74D)
- active state, selected state (cyan glow)

## Shape Descriptions

| Technical | Natural Language |
|-----------|-----------------|
| `rounded-none` | sharp, squared-off edges |
| `rounded-sm` | slightly softened corners |
| `rounded-md` | gently rounded corners |
| `rounded-lg` | generously rounded (16px) |
| `rounded-xl` | very rounded, pillow-like (20px) |
| `rounded-full` | pill-shaped, circular |
